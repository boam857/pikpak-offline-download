## 使用上要注意什麼？

幾個使用前需要知道的實際情況：

- **離線下載並非 100% 成功**：非常冷門、完全沒有做種的 BT 資源有時仍然無法下載，這是 P2P 資源本身的問題，不是 PikPak 的鍋
- **儲存空間不等於本地儲存**：PikPak 的 10TB 是雲端空間，不是下載到你本地設備的空間
- **網頁版在部分地區需要穩定的網路環境**：App 版通常直連更穩定
- **手機息屏可能中斷下載本地**：如果你在 App 內把資源「下載到本地」，息屏後有可能中斷；建議充電狀態進行本地下載。雲端下載任務本身不受影響，因為是在伺服器端完成的
